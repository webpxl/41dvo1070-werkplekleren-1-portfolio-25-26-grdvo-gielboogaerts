Dit is mijn persoonlijke portfolio website.

Ik heb een home-, werk-, about- en contactpagina. Binnen de werkpagina heb ik per project waarbij een uitleg is, een extra pagina. Ook de onzichtbare '/portfolio'-pagina is aanwezig.
Elke afbeelding die gebruikt wordt is van mezelf afkomstig en rechtenvrij. 

Gevraagde inhoud:
Korte voorstelling - aanwezig
(Minstens) 3 (school)projecten - aanwezig
'Speciale' sectie - aanwezig in de vorm van de interactieve wereldkaart

De website is volledig zelf gecodeerd, buiten de gebruikte javascript (interactieve wereldkaart, 'Go to top'-knop en bewegingsanimatie op de werkpagina).
